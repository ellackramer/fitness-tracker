# fitness-tracker
homework 17 fitness tracker app developed using Node and MongoDB.

# installation 
requires NodeJS and MongoDB to run successfully

to install:
    git clone https://github.com/ellackramer/Fitness-Tracker.git

    cd Fitness\ Tracker/

    npm install

    npm start
